# DuckFlix Website

> The official landing page for [DuckFlix](https://github.com/duckflixapp/duckflix) — an open-source, self-hosted media platform.

This repository contains the source code for the official DuckFlix website, featuring a clean, minimalist design with full dark/light mode support and smooth animations

## Tech Stack

- **Framework:** React + TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
